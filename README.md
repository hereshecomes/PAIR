# PAIR

PAIR is an anonymized research code skeleton for paired intervention evaluation of tool-using agents.

This release intentionally contains only the core library components needed to inspect the benchmark construction, environment interfaces, canonical trace processing, and programmatic scoring logic.

Excluded from this anonymous package: experiment runners, baseline implementations, generated datasets, raw traces, model outputs, reports, paper drafts, cache files, and machine-specific configuration.

Install locally with:

```bash
pip install -e .
```
